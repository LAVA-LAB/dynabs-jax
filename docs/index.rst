DynAbs-JAX documentation
========================

This repository contains a Python implementation of the techniques proposed in the paper:

    *Thom Badings & Alessandro Abate (2025). Probabilistic Alternating Simulations for Policy Synthesis in Uncertain Stochastic Dynamical Systems.*

This code can be used to reproduce the experimental results from the paper.
